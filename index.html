CocoSorter

CocoSorter is an automated sorting system using a Raspberry Pi and Arduino. It classifies coconuts by analyzing color for ripeness with a camera and internal water content via audio from a microphone, improving grading consistency and reducing manual labor.

  • Python
  • Arduino
  • Raspberry Pi
  • Hardware Integration
  • Image Recognition
  • Audio Processing
